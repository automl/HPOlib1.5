class hpolib.benchmarks.synthetic_functions.forrester.Forrester[source]¶

Bases: hpolib.abstract_benchmark.AbstractBenchmark

one dimensioal function with one global minimum, another local minimum and a zero gradient, inflection point.

Simple 1d function for visualization. We added a different ‘fidelity’ than the listed Sources. We interpolate between the original function and a Least-Square cubic fit.

One can add a an arbitray cost model externaly to simulate true multi-fidelity scenarios.

http://www.sfu.ca/~ssurjano/forretal08.html

Forrester, A., Sobester, A., & Keane, A. (2008). Engineering design via surrogate modelling: a practical guide. Wiley.
