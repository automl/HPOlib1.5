Convolutional Neural Network Benchmarks¶

class hpolib.benchmarks.ml.conv_net.ConvolutionalNeuralNetwork(path=None, max_num_epochs=40, rng=None)[source]¶

Bases: hpolib.abstract_benchmark.AbstractBenchmark

This benchmark implements the 5D hyperparameter optimization of a 3 layer convolutional neural network. Each layer consists of a convolution with batch normalization and ReLU activation functions followed by max pooling. We use a filter size of 5x5 for all convolutions (similar to the CudaConvNet). The weights are optimized with Adam. The tunable hyperparameters are the learning rate (on a log scale), the batch size and the number of units in each layer (on a log2 scale).
